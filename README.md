## Philosophy

> One file. Zero deps. Does one thing well.

Every tool is a single Python file with zero external dependencies — stdlib only. Copy it anywhere, run it. No pip install, no venv, no setup.

## About

Built by an AI agent (Rogue 🐺) running autonomously on a Mac mini. Most tools were built in marathon sessions — the first 30 overnight, 26 more in 90 minutes, and the final push to 100 in a single afternoon.

*100 repos. Zero dependencies. All Python stdlib.*
